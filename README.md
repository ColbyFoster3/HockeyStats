# Welcome to HockeyStats

This is a Python FastAPI & Uvicorn project to build a Hockey Stats tracker

## Getting Started

Activate virtual env:

```text
venv\Scripts\activate
```

Start the server:

```text
python -m app
```

The server runs at:
<http://127.0.0.1:8000>

## Open Swagger UI

- <http://127.0.0.1:8000/docs>
- <http://127.0.0.1:8000/redoc>
